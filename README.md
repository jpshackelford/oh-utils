# OpenHands Utilities

A collection of utilities for working with OpenHands (formerly All-Hands-AI).

## Utilities

### [OpenHands API Utility](./openhands-api-utility/)

A Python utility for managing OpenHands conversations via the API. Features include:

- Secure API key management
- List and browse conversations with pagination
- Download workspace archives
- Download changed files from conversations
- Interactive command-line interface

See the [OpenHands API Utility README](./openhands-api-utility/README.md) for detailed usage instructions.
